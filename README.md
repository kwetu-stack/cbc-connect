# CBC-Connect v2

CBC-Connect v2 is a lightweight Teacher Observation & Reporting System designed for CBC (Competency-Based Curriculum) environments.

It enables teachers to record learner observations, track skills development, and generate weekly summaries and reports in a clean, distraction-free interface.

---

## Features

- Secure teacher login
- Class and learner navigation
- Observation recording (activity, skill, level, notes)
- Weekly summary dashboard
- Observations index
- Reports table view
- Clean Kwetu Partners UI styling
- SQLite database (local-first, demo-ready)

---

## Tech Stack

- Python (Flask)
- SQLite
- Jinja2 templates
- Vanilla CSS (Kwetu Partners styling)
- No frontend frameworks
